Given two strings s1 and s2, write a function to return true if s2 contains the permutation of s1. In other words, one of the first string's permutations is the substring of the second string.

 

Example 1:

Input: s1 = "ab" (pattern) s2 = "eidbaooo" (string )
Output: True
Explanation: s2 contains one permutation of s1 ("ba").
Example 2:

Input:s1= "ab" s2 = "eidboaoo"
Output: False

Compute patValue and strValue if they are equal proceed to check their frequency hash table;
1. DSA Used: Hash table
2. Complexity: Easy
3. Time taken: 25 minS
4. Time Comlexity: O(strlen)
5. Space Complexity: O(patLen)
